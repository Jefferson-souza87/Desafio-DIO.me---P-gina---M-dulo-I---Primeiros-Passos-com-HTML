04 - Estrutura Básica do HTML

A Estrutura básica do HTML consiste em 3 partes: HTML - HEAD - BODY

  • html = ‹html› ‹/html›

    • incia o HTML COM SINAL DE MAIOR ‹ e fecha com a barra / e sinal de MENOR ›

  • head = ‹head›‹/head›

    • incia o HEAD COM SINAL DE MAIOR ‹ e fecha com a barra / e sinal de MENOR ›

  • body = ‹body›‹/body›

    • incia o BODY COM SINAL DE MAIOR ‹ e fecha com a barra / e sinal de MENOR ›

O que são essas estruturas :

HTML

É o bloco do código principal

O que são essas estruturas :

HEAD

É a "cabeça" da página, tudo aquilo que configura a página mas não aparece visualmente para o usúario

O que são essas estruturas :

BODY

É o Corpo da página, neste caso tudo que é marcado dentro da tag BODY é apresentado ao usuário visualmente

05 - Falando sobre Tags

A Tag HTML é um tipo de marca que define o conteúdo, ela define o tipo de conteúdo que define como será exibido na página

Exemplos de tags:

  • H1 - H2 - H3 - H4 - H5 - H6

    São tags destinadas aos titulos da página, sendo a H1 como a tag para o titulo princial e as subsequentes como subtiluos

  • P

    a tag P é destinada a identificação de paragráfos.

  • Strong

    a tag STRONG é muito utilizada para dar grande importância a certa palavra

  • A

    a tag A serve para "ancorar" criar um vinculo com outro documento, podendo ser

  • HR

    tag utilizada para criar um linha e dividir o conteudo

  • UL - OL

    As tags UL e OL são tags para criarmos listas ordenadas e não ordenadas:
    OL: lista ordenada, onde segue uma sequencia, muito utlizada para listarmos um receita por exemplo onde devem ser seguido os ingredientes conforme sua ordem:

    1. primeiro acrescentamos a farinha
    2. segundo acrescentamos os ovos
    3. terceiro deve ser colocado o leite
    4. por ultimo mexa a mistura até formar uma massa lisa e consistente

    UL: lista não ordenada, diferente da lista OL ordenada, na UL não temos uma sequencia a ser seguida:

    • Pastel
    • Coxinha
    • Empada
    • Pizza
